<!-- ABOUT THE PROJECT -->
## About The Project

Using this extension, you can extract and copy equations/expressions from Deltamath to your clipboard without it being encrypted.

<p align="right">(<a href="#readme-top">back to top</a>)</p>


<!-- GETTING STARTED -->
## Getting Started

1. Download the Chrome extension as a ZIP file from the GitHub page
2. Extract the ZIP file

### Installation

1. Go to the Extensions tab
2. Click on "Manage extensions"
3. Turn on Developer Mode on the top-right
4. Click on "Load unpacked" on the top-left
5. Select the folder of the ZIP you just extracted

<p align="right">(<a href="#readme-top">back to top</a>)</p>


<!-- USAGE EXAMPLES -->
## Usage

To use the extension, click on the Extensions tab and pin it for ease of use.

Click on the extension when an equation/expression is open in the page, and press on "Copy Equation" to copy the equation to your clipboard.

<p align="right">(<a href="#readme-top">back to top</a>)</p>
